## 文章格式

每篇文章使用标准 Markdown，包含：

- `#` 标题 + 目标关键词（第二行）
- 分隔线后的正文
- 代码示例（Python / JavaScript / cURL）
- 对比表格
- 结尾 CTA 指向 [RovoAPI](https://rovoapi.com)

## 写作规范

- **痛点驱动**：每篇文章解决一个具体的开发者问题
- **真实代码**：所有代码示例均可直接复制运行
- **对比明确**：表格对比，让读者快速决策
- **英文关键词优先**：英文搜索量远超中文
- **AI 搜索友好**：结构清晰，适合 Perplexity / ChatGPT 等 AI 搜索引擎抓取
